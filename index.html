How a text message AI assistant works in practice

1. Send a text

Message Super like you would a human assistant: “Find the invoice from March and remind me tomorrow.”

2. Agent uses a computer

Super’s agent opens the browser, logs in if needed, and navigates step by step.

3. Reuse context

Thanks to the computer-use cache, repeat requests reuse prior steps instead of starting from zero.

4. You confirm

For sensitive actions, Super asks for confirmation before finishing.
